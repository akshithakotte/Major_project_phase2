🔹 Model Training and Evaluation

The proposed system uses a deep learning–based CNN model for detection. The dataset is first preprocessed by normalization and label encoding. The dataset is split into training and testing sets to ensure unbiased evaluation. The CNN model is trained for 20 epochs using the training dataset. During training, the model learns discriminative features from the input data to accurately classify instances.

🔹 Performance Metrics

After training, the model is evaluated using the test dataset. Accuracy is used as the primary performance metric to measure the overall correctness of predictions. Additionally, a confusion matrix is generated to analyze class-wise prediction performance. The confusion matrix helps in identifying false positives, false negatives, true positives, and true negatives.

🔹 Confusion Matrix Analysis

The confusion matrix visually represents the classification results, where diagonal elements indicate correct predictions and off-diagonal elements indicate misclassifications. This analysis provides deeper insights into the strengths and weaknesses of the proposed detection system.

🔹 Deployment

The trained model is saved and integrated with a web-based application (app.py) that allows users to upload data and perform real-time detection using the trained CNN model.
